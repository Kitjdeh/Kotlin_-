## 사용기술
Room
내부 DB 관리
Coroutine
네트워크 리퀘스트 및 내부 저장소 접근
Datastore
사용자 재접속 판별을 위한 내부 저장소
Lottie
앱 실행시 로딩 스피너에 애니메이션 효과
Retrofit2
코인 데이터 통신
Navigation
프레그먼트간의 이동
Timber
릴리즈 모드에서 로그가 남지 않도록 설정
SpalshScreen
로딩 화면 설정
Livedata
LifeCycle 확인을 통한 데이터 및 UI 유지
Workmanager
백그라운드에서도 누적 데이터 확보를 위한 실행 예약
ForegroundService
사용자 Notification 생성
